# Ui-Clone-Nubank
Clonagem das telas principais do app do nubank

- baixe o projeto
- cd ui-clone-nubank
- yarn install ou npm install
- yarn start ou npm start

<hr />

### Features e ferramentas usadas

- [x] React Native/Expo
- [x] Styled-Components
- [x] TypeScript
